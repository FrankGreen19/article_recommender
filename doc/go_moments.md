# Указатели 

## Возвращать ссылки на объект стоит в случаях:
- если объект крупный (сордержит большие массивы, срезы);
- нужно изменять оригинал (например, при сохранении в БД).

## Принмать ссылки на объект стоит в случаях:
- функция должна изменить этот объект;
- структура тяжёлая (внутри большие поля);
